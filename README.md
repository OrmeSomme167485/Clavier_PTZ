# Clavier V1 pour Cam PTZ 

Pour utiliser le clavier il faut installer deux choses sur le pc:

1 Python 
https://www.python.org/downloads/

2 Les bibliothèques avec la commande :

    py -m pip install pyserial pyautogui 
    

Pour vérif que tous est bien installé

1 Vérif que python est bien installé 

    python --version 

2 Vérif que pip fonctionne 

    pip --version

3 Vérif des bibliothèques (PyAutoGUI)et(pyserial)

    pip list
